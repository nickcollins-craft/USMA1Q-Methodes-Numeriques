# USMA1Q — Méthodes Numériques

Cours de méthodes numériques pour le diplôme d'ingénieur en spécialité Matériaux du **Conservatoire National des Arts et Métiers**.

Ce cours introduit les outils de programmation Python et de calcul numérique utiles aux ingénieurs en science des matériaux.
