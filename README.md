# USMA1Q — Méthodes Numériques

Cours de méthodes numériques pour le diplôme d'ingénieur en spécialité Matériaux du **Conservatoire National des Arts et Métiers**.

Ce cours introduit les outils de programmation Python et de calcul numérique utiles aux ingénieurs en science des matériaux.

## Contenu du dépôt

### [`presentations/`](presentations/)

Supports de cours sous forme de fichiers .html (diapositives) :

- **Séance 1** — Fondamentaux de Python : variables, types, expressions, structures de contrôle (`if`, `for`, `while`), premières manipulations de données.

### [`exercices/`](exercices/)

Notebooks d'exercices accompagnant chaque séance.

### [`donnees/`](donnees/)

Fichiers de données utilisés dans les exercices.
